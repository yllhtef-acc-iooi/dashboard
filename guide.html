完成後的日常作業，只有一件事

其他人（顧問、董事、同仁）只需要收藏連結，需要的時候開啟即可。
  • 有問題可在網頁的「問與答」功能送出，由同仁人工回覆（寄到 operation@ 和 account@）

  • 想了解整體架構，點「作業說明」開啟本頁

升級所需步驟（共 3 步，預估 30 分鐘）
B1
取得 Anthropic API Key
前往 console.anthropic.com 登入後，在「API Keys」建立一個新的 Key。
  • Key 長得像：sk-ant-api03-xxxxxxxxxx…
  • 建立後請妥善保存，只會顯示一次
  • 費用：依使用量計費，一般提問量每月約 1–5 美元
待辦
B2
建立 Cloudflare Worker（免費中轉服務）
前往 workers.cloudflare.com 免費註冊，建立一個 Worker，貼上事先準備好的程式碼，填入 API Key。
  • 完全免費：每天 10 萬次請求，遠超實際使用量
  • Worker 會得到一個固定網址，例如：https://yllhtef-qa.workers.dev
  • 程式碼由 Claude 協助產生，不需要自己寫
待辦
B3
更新儀表板 HTML，指向 Worker 網址
在儀表板 HTML 中，將問與答的 API 呼叫目標改為 Cloudflare Worker 的網址，重新上傳到 GitHub 即完成。
  • 只需修改一行程式碼：var QA_WORKER_URL = 'https://yllhtef-qa.workers.dev';
  • 上傳新版 index.html 覆蓋舊版，網址不變
  • 升級後「問與答」即可即時回答，Formspree 收件功能仍保留作備援
待辦
💡 準備好升級時，只需告訴 Claude：「我要進行方案 B 升級」，並提供 Anthropic API Key 和 Cloudflare Worker 網址，Claude 會協助產生程式碼並更新儀表板。
